# Notes Checklist Backend

Simple Express + MongoDB backend scaffold for Notes + Checklist app.

Run:

```bash
cd backend-project
npm install
npm run dev
```

API endpoints:
- Notes: `/api/notes`
- Tasks: `/api/tasks`

Env variables:
- `PORT` - default 5000
- `MONGO_URI` - MongoDB connection string
https://notes-checklist-app.vercel.app/
